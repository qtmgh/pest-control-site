# Snap Rodent and Pest Control Website

A single page website designed for Snap Rodent and Pest Control. The project focuses on simple navigation, strong visuals, and a layout suited for users with short attention spans. The content highlights services, pricing, social media, and quick contact options.

## Features
- Hero section with visuals
- Clear call to action buttons
- Service categories with visual cards
- Simple pricing section
- Contact form with essential fields
- Social media section with TikTok, Instagram
- Customer reviews and before or after examples
- Mobile responsive design

## Project Purpose
This project provides a modern site for the business owner while offering learning experience in:
- Front end structure and layout planning
- Visual hierarchy
- Responsive design
- Client requirements and workflow

## Tech Stack
- HTML
- CSS
- Javascript
